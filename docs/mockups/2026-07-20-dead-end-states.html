Three states that today give almost nothing. Death: a red vignette + a brief "YOU DIED" that names the killer and the XP lost to the level floor, then fades as you respawn — the only current feedback is a scrolling "NAME died" chat line and a silent camera cut. World reset: today a bare location.reload() dumps you on the start screen with no word that your character is gone; this says so. Server down at load: today a tiny dim "failed to start"; this retries with a visible count instead of a dead black page.
