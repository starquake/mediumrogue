Blink shown in all three states — unlearned, ready, cooling. Learn spends a skill point to acquire, exactly as for a passive; Use triggers, and only ever appears once learned. The earlier draft skipped the unlearned row, which made Learn look like it selected the skill. An active badge marks the kind before you have it, so the choice to spend a point is informed.

Reachable hexes lit blue; the hovered destination brighter. Rock (▲) is not lit — Blink needs line of sight, so cover stays real. Click to commit, Esc or B again to cancel.
Colour choice worth challenging: actives use blue against the game's green. It separates "trigger this" from "learn this" at a glance, but it is the first non-green accent in the UI — say if you'd rather keep one palette and distinguish actives by shape alone.
